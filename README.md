## Sobre o Aplicativo
É um aplicativo Android simples para gerenciamento financeiro pessoal, nele você consegue comparar o quanto você ganha em relação ao tanto que você gasta e com isso consegue ver quanto sobra para investir ou quanto precisa economizar.

## Funcionalidades

- **Dashboard de Saldo**: Visualização clara do saldo total com feedback visual de cores (Verde para positivo, Vermelho para negativo).
- **Gerenciamento de Transações**: Adição e remoção de receitas e despesas.
- **Categorização Automática**: Lista organizada que separa automaticamente o que é entrada (Receita) do que é saída (Despesa).
- **Interface Fluida**: Títulos de seções fixos (Sticky Headers) para facilitar a navegação em listas longas.
- **Persistência de Dados**: Seus dados ficam salvos no dispositivo mesmo após fechar o aplicativo.

## Demonstração Visual

- **Dashboard**: Painel superior com borda robusta e saldo destacado.
- **Lista**: Itens com contorno preto e cores semânticas (Verde/Vermelho).
- **Sticky Headers**: Títulos "RECEITAS" e "DESPESAS" que grudam no topo durante a rolagem.

## Como executar o projeto

1. Clone este repositório.
2. Abra o projeto no **Android Studio (Ladybug ou superior)**.
3. Certifique-se de ter o SDK 35 instalado.
4. Execute o app em um emulador ou dispositivo físico com Android 7.0 (API 24) ou superior.
